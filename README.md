# Onibag

Tech task for Onibag.

### Install

Clone this repo

````
git clone https://github.com/Be-Smart/onibag.git
````

Enter in the downloaded folder

````
cd onibag
````

Install dependencies

````
npm install
````

### Usage

Run webpack server

````
npm run dev
````

Now you are ready to test app on adress: http://localhost:8080
